### Instructions

Due to the size limitation in the Online-Campus the models Folder will be empty
Download it from Github or this Link: https://www.dropbox.com/s/v6tcafc1i8rhfff/models.zip?dl=0

1. Install Python 3 (last version)
2. Download the code and unzip it
3. Open the code folder form your shell

4. Installing virtualenv
## On Windows: 
py -m pip install --user virtualenv

## On macOS and Linux:
python3 -m pip install --user virtualenv

5. Creating a virtual environment
## On Windows:
py -m venv env
## On macOS and Linux:
python3 -m venv env

6. Activating a virtual environment
## On Windows:
env\Scripts\activate
## On macOS and Linux:
source env/bin/activate

7. Install the python libraries
Follow this Video: 
https://www.youtube.com/watch?v=xaDJ5xnc8dc&ab_channel=Pythoholic

8. Run the programm
## On Windows:
py main.py
## On macOS and Linux:
python main.py
